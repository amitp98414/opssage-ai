# 🚀 Enterprise AI DevOps Assistant

A production-ready AI-powered DevOps Assistant built with FastAPI, Docker, PostgreSQL, Redis, and OpenAI.

## 🛠 Tech Stack

- Python 3.14
- FastAPI
- Docker
- Docker Compose
- OpenAI API
- PostgreSQL (Coming Soon)
- Redis (Coming Soon)
- GitHub Actions (Coming Soon)

## Features

- FastAPI REST API
- Dockerized Backend
- Health Check Endpoint
- Swagger API Documentation
- Production Folder Structure

## Run

```bash
docker compose up
```

Swagger:

```
http://localhost:8000/docs
```
